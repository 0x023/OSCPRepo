Enabling VNC (from command line)
  Go get VNC (check out  guh.nu!)
  Make a folder on the target for the vnc files
  Copy the following files to target folder:
  Winvnc.exe
  Vnc.reg
  Vnchooks.dll
  Omnithread_rt.dll
  Regedit -s vnc.reg
  Winvnc -install
  Net start "vnc server"
