Link to Passwords and Hashes > Password Locations > Windows > Passwords in Registry 

There are several places you may be able to obtain passwords from in Registry

Auto Logon
Can be set via: Group Policy Preferences, Netplwiz.exe, Editing the registry directly
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon

PuTTY
Proxy credentials
HKEY_CURRENT_USER\Software\<USER>\PuTTY\Sessions

VNC
Location is dependant on the VNC product
Tight VNC: HKCU\Software\TightVNC\Server
reg query "HKCU\Software\ORL\WinVNC3\Password"

SNMP Parameters:
reg query "HKLM\SYSTEM\Current\ControlSet\Services\SNMP"


# Similarly the two commands below can be used to grep the registry for keywords, in this case "password".
 reg query HKLM /f password /t REG_SZ /s
 reg query HKCU /f password /t REG_SZ /s

 reg query HKLM /f passw /t REG_SZ /s
 reg query HKCU /f passw /t REG_SZ /s

# Additional
 reg query HKLM /s | findstr /i password > temp.txt
 reg query HKCU /s | findstr /i password > temp.txt
