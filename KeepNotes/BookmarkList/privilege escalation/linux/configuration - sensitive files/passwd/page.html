Writing to sensitive files: /etc/passwd
Use openssl passwd --help (to show options)
openssl passwd <password> #gives encrypted passwd in stdout

Now add the encrypted password to a user in /etc/passwd (overwrite someone elses password like root!)
You can now login with that password


or just create a new user with UID and GID 0:
openssl passwd -1 -salt ippsec hackthebox
