dev/tcp can be used as a rudimentary web client

exec 3<>/dev/tcp/www.google.com/80
echo -e "GET / HTTP/1.1\r\nhost: http://www.google.com\r\nConnection: close\r\n\r\n" >&3
cat <&3



Note:
some systems (Debian included) don't enable /dev/tcp.
you can use /dev/udp too
