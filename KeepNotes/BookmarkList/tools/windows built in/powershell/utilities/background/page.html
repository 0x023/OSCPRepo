Run a command in the 'background' by starting a new process (detached from main window)

Start-Process -NoNewWindow ping google.com

or here's a useful function:

function bg() {Start-Process -NoNewWindow @args}

and now you can just type:
bg ping google.com



Pros and cons to both Start-Process and Start-Job

Start-Process:
Will continue even if current window/session closes
Unable to redirect commands to a file, ie ping google.com > results.txt does not work

Start-Job
Killed when current window does
Redirection is allowed
Path should be fully qualified as the cwd may change when Start-ob is  called
