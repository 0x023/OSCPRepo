The PowerShell ingestor BloodHound.ps1, implements the Invoke-BloodHound function for collecting and exporting data.

Select the module: Powershell/situation_awareness/network/bloodhound

Select the appropriate download location

Run the module:

CSVs are exported to the defined location

Notice the BloodHound module in Empire uses the function Invoke-BloodHound. This function executes collection options for BloodHound data. You can specify collection options with -CollectionMethod, or specify an export location with -CSVFolder. The default behavior will gather unrolled group membership information, domain trust information, and all session/local admin data from your current domain.
