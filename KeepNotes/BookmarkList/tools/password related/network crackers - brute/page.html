Spray (https://www.trustwave.com/Resources/SpiderLabs-Blog/Simplifying-Password-Spraying/) | A Password Spraying tool for Active Directory Credentials by Jacob Wilkin(Greenwolf) - SMB, OWA, Lync, CISCO Web VPN. This script will password spray a target over a period of time It requires password policy as input so accounts are not locked out. Accompanying this script are a series of hand crafted password files for multiple languages. These have been crafted from the most common active directory passwords in various languages and all fit in the complex (1 Upper, 1 lower, 1 digit) catagory.
