Just a quick note, here is a list of vulnerable PHP functions that you should look out for in your web apps:
Local / Remote file inclusion bugs:

include()
include_once()
require()
require_once()

Local / Remote command execution bugs:
eval()
preg_replace()
fwrite()
passthru() Also sends binary data, but requires a bit more work
file_get_contents()
shell_exec() Returns every line
system()

SQL Injection bugs:
mysql_query()

File / File system bugs:
fopen()
readfile()
glob()
file()
popen()
exec() Returns only the last line
