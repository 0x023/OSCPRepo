PHP:
Typically, cmd shells use $_GET. $_GET uses args in the URL.
$_POST typically uses args in the body of the POST request
$_REQUEST can take args in either the body or the URL. Helps when doing things through BURP

IFS = Internal Field Separator
IFS is 'space' by default, so certain commands will split based on IFS

Example: ps -eo command #list commands of processes
when used in a loop, will print each command separated by spaces

for i in $(ps -eo command); do echo $i; done
start
/usr/sbin/apache2
-k
etc etc etc

Now define IFS as something else, like "\n": IFS=$'\n'
run the same loop:
for i in $(ps -eo command); do echo $i; done
/usr/sbin/apache2 -k start
etc
