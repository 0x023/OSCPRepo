Hashcat cracking LM, brute both sides of the hash. start with 7 and decrement until you have both sides
?u?u?u?u?u?u?u
?u?u?u?u?u?u
?u?u?u?u?u
etc

?u is hashcat for bruteforcing a capital letter (which LM is all caps)

Take cracked LM hash in both upper and lower, put them in a dictionary, now crack the NTLM:
hashcat -m 1000 ntlm.hash lm.dict -r rules/toggles5.rule

the rule will toggle casing. since LM is all upper, it is usually NOT the actual password, but NTLM will have actual password with casing
