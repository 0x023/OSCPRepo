Knock on ports in a specific order in order to open up a new port

/etc/init.d look for knockd (port-knock daemon)
Default config usually
 /etc/default/knockd
or
 /etc/knockd.conf


for i in knock port numbers; do nmap -Pn -p $i --host_timeout 201 --max-retries 0 ip_address; done
