 Type 'generate' and Veil-Evasion proceeds to compile my executable. In addition to creating my .exe, you’ll notice below that a payload file and a handler file (.rc) have also been generated and dumped under /var/lib/veil-evasion/output/. The handler file is convenient, as running it with the following syntax will fire up Metasploit and set up all the appropriate handler options on your behalf:

 msfconsole -r /var/lib/veil-evasion/output/handlers/notepad_handler.rc
