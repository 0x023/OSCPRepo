The definition of the event to trigger
Takes the form of a WMI query
Be mindful of performance!
These take some practice...

Intrinsic query:
SSELECT * FROM __InstanceOperationEvent WITHIN 30 WHERE ((__CLASS = "__InstanceCreationEvent" OR __CLASS = "__InstanceModificationEvent") AND TargetInstance ISA "CIM_DataFile") AND (TargetInstance.Extension = "doc") OR (TargetInstance.Extension = "docx")


Extrinsic query
SELECT * FROM Win32_VolumeChangeEvent WHERE EventType = 2
