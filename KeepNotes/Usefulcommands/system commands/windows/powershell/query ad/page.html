Get Information from a RODC
Get-ADComputer $ROCName -Property * | Select Name,ManagedBy,’msDS-AuthenticatedToAccountlist’,’msDS-NeverRevealGroup’,’msDS-RevealedDSAs’,’msDS-RevealedUsers’,’msDS-RevealOnDemandGroup’v

Get list of users with passwords cached on a RODC
<RODC>.'msDS-RevealedUsers' | % {(_-split(':'))[3]} | sort | sort -Unique
