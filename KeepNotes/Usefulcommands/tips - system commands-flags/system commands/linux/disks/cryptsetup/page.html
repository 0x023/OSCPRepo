sudo cryptsetup luksFormat </dev/sda1> encrypt a partition
cryptsetup open </dev/sda1> backup mount encrypted partition part 1
mount </dev/mapper/backup> </media/username/back> mount encrypted partition part 2

sudo cryptsetup luksDump </dev/sda3> view passphrase slots
sudo cryptsetup luksAddKey </dev/sda3> add a key
sudo cryptsetup luksRemoveKey </dev/sda3> remove key
